# Sentiment Analysis - UAE Customer Reviews

## Project Overview
This project performs sentiment analysis on UAE customer reviews using Natural Language Processing (NLP).

## Steps
- Data cleaning and preprocessing
- TF-IDF vectorization
- Logistic Regression model training
- Model evaluation using confusion matrix and classification report
- WordCloud and LDA topic modeling

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- NLTK
- Matplotlib, Seaborn

## Model Performance
- Logistic Regression used for classification
- Evaluated using accuracy, precision, recall, F1-score

## Output
- Sentiment predictions saved in CSV
- Trained model saved using joblib
